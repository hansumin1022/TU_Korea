### ★ Infineon AURIX TC277 보드 data sheet 요약

| MCU | Infineon AURIX TC277|
|--|---|
| 아키텍처 | <b>TriCore™ (32-bit)</b> (TC1.6P 2개 + TC1.6E_저전력 1개) |
| 클럭 수 | 최대 <b>200 MHz</b> |
| 동작 온도 | -40~125 °C 수준 |
| 메모리 | Flash 최대 4 MB |
| 지원 가능 통신 인터페이스 | CAN-FD, FlexRay, Ethernet |
| ASIL | <b>ISO 26262 ASIL-D</b> 등급 지원 |
